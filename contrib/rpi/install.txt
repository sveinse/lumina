
1. Download image from here:

https://www.raspberrypi.org/downloads/raspbian/


2. Unpack and install

   $ unzip 2016-05-27-raspbian-jessie-lite.zip
   $ sudo dcfldd bs=4M if=2016-05-27-raspbian-jessie-lite.img of=/dev/mmcblk0


3. Start on target


4. Login

   user: pi
   password: raspberry

   Become root


5. Run raspi-config

   # raspi-config

      - Advanced, update tool to latest

      - Set pi password
      - Add locale, default en_US.UTF-8
            en_US.UTF-8
            nb_NO.UTF-8
      - Set TZ
      - Set KBD
      - Set Wifi country
      - Set hostname (lum2)
      - Enable ssh
      - Enable wait on network (which is default)


6. Update

   # apt update
   # apt dist-upgrade


7. Setup WiFi

   # nano /etc/wpa_supplicant/wpa_supplicant.conf

   network={
        ssid="testing"
        psk="testingPassword"
   }


8. Install packages

   # apt install rsync


9. Update boot config

   # nano /etc/config.txt

   hdmi_group = 1
   hdmi_mode = 16    # 16=1080p, 4=720p
   #dtparam=audio=on


10. Install pip

   # apt install python-pip
